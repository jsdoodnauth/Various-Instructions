##Layers
- New Layer -> Shift-Opt-Cmd+N
- Stamp Visible Layer on top -> Ctrl-Alt-Shift-E

##Sharpen

- Blending Mode -> Soft Light
- Opacity -> 50%
- New Layer
- Desaturate -> Ctrl-Shift-U
- Blending Mode -> Overlay
- High Pass -> (3.5 pixels)
